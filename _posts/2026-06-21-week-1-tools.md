I spent the first weekend getting my Mac set up properly. I installed Apple’s Command Line Tools, Homebrew, VS Code and its Python extension, and Miniconda. I added Homebrew to my `PATH`, switched conda and pip to the Tsinghua mirrors, set up SSH access to GitHub, and made a test push. I also got this blog running on GitHub Pages.

Monday and Tuesday were shell and Git days. I worked through the first two Missing Semester lectures and their exercises, then wrote my own shell cheatsheet. I spent some time on Learn Git Branching too, then kept practicing until I could run through the Git workflow without my notes. Git felt a lot less mysterious after that.

On Wednesday I did a quick Python self-check, filled a few gaps, then wrote a small study log that saves data as JSON, times methods with a decorator, and calculates weekly statistics. Thursday was NumPy. I made it through the first 50 exercises in numpy-100, focusing on vectorization and broadcasting.

On Friday I tried both `pdb` and VS Code’s debugger, getting used to stepping through a program and inspecting variables.

On Saturday I finally put the setup to use. I made a clean environment for CleanRL, ran DQN on `CartPole-v1`, watched the return curve in TensorBoard, then changed the learning rate and training length and compared the runs. It was nice to finish the week with an actual training curve on the screen.

I kept up with linear algebra as well, working through the 3Blue1Brown series from vectors and linear transformations to eigenvectors, and checking several of the ideas with NumPy as I went.

Next week I’m starting Karpathy’s Zero to Hero. I’ll begin with micrograd, then move on to makemore and a small MNIST model. I want to understand backpropagation by building it myself before I rely on PyTorch.
